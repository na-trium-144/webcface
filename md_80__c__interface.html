Interface for C

API Reference → webcface/c_wcf.h

CからアクセスできるAPIとして、wcf〜 で始まる名前の関数やstructが用意されています。 <webcface/c_wcf.h> をincludeすることで使えます。

ほとんどの関数は戻り値がint型で、成功した場合0(=WCF_OK)、例外が発生した場合正の値を返します。
